LEADTOOLS OCR Spell Check Engines

Using a spell checker with the LEADTOOLS OCR engine is easy. All you need to do is make sure the required prerequisites are available in the system and then set the spell check engine desired in the IOcrSpellCheckManager.SpellCheckEngine property. Setting a spell checker will greatly enhance the OCR accuracy and result in a document that more closely resemble the original text.

LEADTOOLS OCR Engine supports the following spell check engines:

None

No spell checking is performed.

Native

This mode uses LEAD's own proprietary Native spellchecker.

Language Support

English, Spanish, French, German, Italian, Bulgarian, Catalan, Czech, Danish, Greek, Hungarian, Indonesian, Lithuanian, Latvian, Dutch, Norwegian, Polish, Portuguese, Romanian, Russian, Slovak, Slovenian, Serbian, Swedish, Thai, Ukrainian, Vietnamese.

Prerequisites

File(s) Description

Dictionary Files

Language dictionary files, these files names must match the equivalent LEADTOOLS LEAD .bin files located at the LEADTOOLS OCR runtime folder but they must have the extension .dic.

The dictionary file must reside in the LEADTOOLS OCR runtime folder. By default, the location is:

"(LEADTOOLS Installation folder)\Bin\Common\OcrLEADRuntime"

OS

The OS mode uses the Microsoft .NET spellchecker.

Language Support

English, French, German and Spanish.

Prerequisites

File(s) Description

Leadtools.SpellChecker.OS.dll

LEADTOOLS support for .NET spell checker. This assembly must be in your Bin folder next to Leadtools.Ocr.LEADEngine.dll.

Microsoft .NET Runtime 3.0 or later

The .NET spell check support is included in the Microsoft .NET runtime version 3.0 or later. Make sure you have this installed on your machine.

Hunspell

This mode uses the popular Hunspell spell check engine (through the standard .NET NHunspell wrapper). LEADTOOLS setup does not contain the NHunspell binaries or any language dictionaries. You must download the required assemblies from the NHunspell page as well as the dictionaries for the languages you are interested in. Visit the NHunspell home page at http://nhunspell.sourceforge.net for more information.

Language Support

Any language supported by both LEAD OCR and NHunspell. This includes: English, Spanish, French, German, Italian, Bulgarian, Catalan, Czech, Danish, Greek, Finnish, Hungarian, Indonesian, Lithuanian, Latvian, Dutch, Norwegian, Polish, Portuguese, Romanian, Russian, Slovak, Slovenian, Serbian, Swedish, Turkish, Ukrainian, Vietnamese, Japanese, Korean and Chinese (Simplified and Traditional).

Prerequisites

File(s) Description

Leadtools.SpellChecker.Hunspell.dll

LEADTOOLS support for NHunspell spellchecker. This assembly must be in your Bin folder next to Leadtools.Ocr.LEADEngine.dll.

NHunspell.dll

NHunspell .NET wrapper. Must be in your bin folder next to Leadtools.SpellChecker.Hunspell.dll.

Hunspellx86.dll or Hunspellx64.dll

Hunspell native DLL. The correct version for your platform must exist in the LEAD OCR runtime folder. By default, the location is:

"(LEADTOOLS Installation folder)\Bin\Common\OcrLEADRuntime"

lang.aff and lang.dic dictionary files

Language dictionary files for the Hunspell engine. Visit http://wiki.services.openoffice.org/wiki/Dictionaries to download the files for the languages you are interested in. You can use the same name as the original dictionary file, for example, "en_US" for English US dictionary.

The dictionary file must reside in the LEAD OCR runtime folder. By default, the location is:

"(LEADTOOLS Installation folder)\Bin\Common\OcrLEADRuntime"
